Politique de Confidentialité

1. Introduction

Bienvenue sur Vylinkia Invest. Nous respectons votre vie privée et nous nous engageons à protéger vos données personnelles. Cette politique de confidentialité vous informe sur la manière dont nous traitons vos données lorsque vous utilisez notre application mobile et notre site web.

2. Données collectées

Nous collectons un minimum de données pour assurer le bon fonctionnement de l'application :

  • Identifiants d'appareil (pour les notifications push).
  • Données d'utilisation anonymisées (pour améliorer nos services).
  • Informations de compte (si vous créez un compte).

3. Utilisation des données

Vos données sont utilisées uniquement pour :

  • Vous fournir les signaux de trading et les alertes.
  • Gérer votre abonnement et votre accès aux fonctionnalités premium.
  • Améliorer la qualité de notre service.
